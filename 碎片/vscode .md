# vscode 快捷键

##  跳转到指定行

Ctrl + G

## 跳转到文件头部

Ctrl + Home

## 跳转到文件尾部

Ctrl + End

## 移动行

Alt + up/down

## 显示隐藏目录栏

Ctrl + b

## 新建一个窗口

Ctrl + Shift + k

## 行向左移动

Ctrl + [

## 行向右移动

Ctrl + ]

## 删除当前行

shift + ctrl +  k
